Data and Software

Framewise Integrated Real-time MRI Monitoring

Framewise Integrated Real-time MRI Monitoring (FIRMM) is a technology that has been pioneered by several members of the DCAN Lab. Easy to set up and use, the FIRMM software suite provides MRI scanner operators with data quality metrics in real time. Using FIRMM to identify the ideal scan time for each person can reduce total brain MRI scan times and associated costs by 50% or more. Learn more at the FIRMM website
